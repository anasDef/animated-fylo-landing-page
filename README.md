# Fylo Landing Page (Animated)

This project is a modern, responsive landing page for "Fylo," a cloud storage service. It features a clean, professional design with smooth animations to enhance the user experience.

## 🚀 Live Demo
You can view the live project here: 
[https://anasdef.github.io/animated-fylo-landing-page/](https://anasdef.github.io/animated-fylo-landing-page/)

## 🛠 Technologies Used
The project was built using the core web technologies, along with animation libraries:
* **HTML5**: For page structure.
* **CSS3**: For styling and responsive layout.
* **JavaScript**: For email validation logic.
* **AOS (Animate On Scroll)**: To add scroll-triggered animation effects.
* **Remix Icon**: For modern vector icons.

## ✨ Project Features
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
* **Smooth Animations**: Integrated AOS library for engaging, dynamic movement when scrolling.
* **Form Validation**: Real-time email validation with error messaging.
* **Modern UI**: Clean interface with professional color gradients.
* **High Performance**: Well-structured, lightweight code for fast loading times.
